# AnimalAppBot
Проект телеграмм-бота, который сможет отвечать на популярные вопросы людей о том, что нужно знать и уметь, чтобы забрать животное из приюта.
Также волонтеры приюта в течение месяца просят новых хозяев присылать ежедневный отчет о том, как животное приспосабливается к новой обстановке. 
В разработке проекта принимает участие 4 участника.
